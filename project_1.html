Movie Recommender Infobot



The Movie Recommender Know-It-All Chatbot aims to enhance the movie-watching experience by addressing two main challenges: finding personalized movie recommendations and providing accurate answers to movie-related questions. This chatbot allows users to input their favorite genres, directors, producers, and actors to receive tailored movie suggestions. Additionally, users can ask specific questions about movies, such as details about the cast and crew, and get accurate answers.

The chatbot is built using Python 3.11, HuggingFace Transformers with DistilBERT, and a Streamlit interface. It features personalized movie recommendations based on user preferences, individual attribute-based movie recommendations that prioritize certain metrics like genre similarities, and a movie question-answering function. The setup involves cloning the repository, creating a virtual environment, and installing the necessary dependencies. Users can then run the application with Streamlit and interact with the chatbot to get movie recommendations or answers to their movie-related queries.
